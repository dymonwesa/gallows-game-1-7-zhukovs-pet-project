# Gallows Game (Виселица)

Консольная игра «Виселица» на Java. Компьютер загадывает случайное слово из встроенного списка, игрок вводит буквы по одной, пытаясь угадать слово до того, как закончатся попытки.

## Стек

- Java
- Без сборщика (Maven/Gradle не используются) — проект запускается напрямую из IntelliJ IDEA
- Ввод через `Scanner` (консоль)

## Как запустить

1. Открыть проект в IntelliJ IDEA
2. Запустить класс `Main` (кнопка **Run** ▶️)
3. В меню выбрать:
    - `1` — начать игру
    - `2` — выйти

## Как играть

1. Программа загадывает случайное слово из списка (`собака`, `кошка`, `атмосфера`, `диван`, `знак`, `зеркало`)
2. Слово показывается в виде `*` по количеству букв
3. Игрок вводит по одной букве за ход
4. Если буква есть в слове — она открывается на всех своих позициях
5. Если буквы нет — засчитывается ошибка, и рисуется очередная часть виселицы
6. Победа — когда все буквы открыты; поражение — после 6 ошибок

## Структура проекта

```
src/
 ├── Main.java            — точка входа, запускает UIClass
 ├── UIClass.java          — консольное меню и игровой цикл
 ├── GamePattern.java       — логика раунда: слово, сравнение букв, открытие позиций, проверка победы/поражения
 ├── NumberOfMistakes.java  — счётчик ошибок и ASCII-рисунок виселицы по этапам (1–6)
 └── InputLetters.java      — учёт уже введённых букв (чтобы не засчитывать повтор)
```

## Известные особенности / TODO

- Список слов зашит в код (`GamePattern.list`) — можно вынести в файл/конфиг
- Нет проверки на ввод не-буквенных символов, кроме базовой (`Character.isLetter`)
- Ошибка при вводе не-строки (`scanner.nextInt()` / `scanner.next()`) приведёт к исключению — обработка edge-кейсов не реализована
- Регистр букв не приводится к единому виду (заглавные и строчные считаются разными)
